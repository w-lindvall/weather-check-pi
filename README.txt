# WEATHER CHECK

-A simple app used to return temperature and humidity values from DHT11 sensor on Raspberry Pi, along with a captured still

# USAGE

-Ensure app.py uses the corresponding pin number used in your DHT11 setup
-Define suitable path for saving still output
-Run app.py
